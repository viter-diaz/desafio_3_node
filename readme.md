# 📦 Gerador de QR Code para E-commerce

Projeto desenvolvido no desafio da DIO: **Formação Node.js - Projeto QR Code**.

## 🚀 Sobre o projeto

Este projeto é um kit de utilidades para e-commerce feito com Node.js.

Funcionalidades:

- Gerar QR Code pelo terminal
- Gerar senhas seguras
- Utilizar variáveis de ambiente
- Trabalhar com dependências NPM
- Organizar o projeto em camadas

## 🛠️ Tecnologias utilizadas

- Node.js
- JavaScript
- NPM
- Chalk
- Prompt
- QRCode Terminal
